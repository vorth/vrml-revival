July 15, 2023 update.

This page was done in late 1990s when VRML (Virtiual Reality Modeling Language) was a buzz word. In just few years following 2000 dotcom crash VRML becomes mostly dead. All VRML plugins for browsers have disappeared. I was sad and kept the page alive for myself by maintaining the last working VRML plugin Cortona for Internet Explorer. Finally Internet Explorer was removed from Windows 11 and I had no way to display any of my VRML files. Few days ago I've learned about existence of X-ITE - javascript library which is capable to display VRML files in any browser with WebGL support. Initially I was sceptical, but after minor tweaking of the code I found, that X-ITE actually works and is pretty capable. Most of the files on that page now work again (after 20 yeast of not working). I would cal this a miracle!

Now I have a task to adapt 8000+ VRML files scattered on that website.
