Polyhedral Transformations

Several polyhedral families in perpetual kaleidoscopical dance.

(5 3 2)
(3 3 5/2)
(5 5 3/2)
(5 5/2 2)
(5 3 5/3)
(5/2 5/2 5/2)

These polyhedra dance best with Cortona VRML viewer.
I am not aware about other VRML viewers, which would work here.

Update July 16, 2023

Now this page uses javascript library X-ITE to display VRML in the browser using WebGL.
